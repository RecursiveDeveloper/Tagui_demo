## Installation

Use ***TagUI_Install.ps1*** powershell script to install tagui in your Windows O.S . Currently, tagui installer it's intended to work on **Windows O.S** for now. A **Linux** installer is under development. 

```powershell
.\TagUI_Install.ps1
```
